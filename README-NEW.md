## 🎯 Visão Geral

Este projeto é uma homenagem às mulheres negras que transformaram a tecnologia, desde as computadoras da NASA até as empreendedoras e cientistas de hoje. Featuring:

- **Design Moderno**: Interface dark mode com animações suaves e efeitos interativos
- **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Acessível**: Semântica HTML apropriada e contraste de cor adequado
- **Modularizado**: JavaScript organizado em componentes reutilizáveis
- **Temas Personalizados**: Cada mulher tem sua cor e design exclusivos

## 🚀 Como Usar

### 1. Clonar ou Baixar o Repositório
```bash
git clone https://github.com/seu-usuario/Trabalho-Escolar-Cultura-Digital.git
cd Trabalho-Escolar-Cultura-Digital
```

### 2. Servir Localmente
Você pode usar qualquer servidor local. Exemplo com Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Então abra `http://localhost:8000` no seu navegador.

### 3. Usar as Novas Versões
Para usar as páginas melhoradas e modularizadas:
- Homepage: `index-new.html` (ou renomeie para `index.html`)
- Biografias: Use as versões com `-new` no final do nome
- Scripts: Use `animacoes/main-new.js` (modularizado)

## 🏗️ Arquitetura

### CSS
O projeto usa um sistema de variáveis CSS para fácil customização:

```css
:root {
    --cor-bg: #0a0a0f;
    --cor-texto: #e4e4e7;
    --grad-1: #FF6B6B;
    --grad-2: #4ECDC4;
    /* ... mais variáveis ... */
}
```

**Arquivos CSS:**
- `global.css` - Resets, tipografia, base
- `components.css` - Componentes reutilizáveis (buttons, badges, etc)
- `homepage.css` - Estilos específicos da página principal
- `biografia.css` - Estilos para páginas de biografia
- `themes.css` - Temas de cores para cada mulher

### JavaScript (Modular)

Cada funcionalidade esta encapsulada em um módulo:

```
animacoes/
├── main-new.js (orquestrador)
└── modules/
    ├── particles.js (background animado)
    ├── cursor.js (efeito glow)
    ├── navigation.js (scroll suave)
    ├── reveal.js (animações ao rolar)
    ├── effects.js (efeitos 3D)
    └── statistics.js (animações de números)
```

**Vantagens dessa abordagem:**
- Fácil de testar
- Reutilizável
- Sem efeitos colaterais
- Fácil de desativar/ativar funcionalidades

## 🎨 Customização

### Mudar Cores Principais
Edite as variáveis CSS em `design/global.css`:

```css
:root {
    --grad-1: #FF6B6B;    /* Cor primária 1 */
    --grad-2: #4ECDC4;    /* Cor primária 2 */
    --grad-3: #A78BFA;    /* Cor primária 3 */
    /* ... */
}
```

### Adicionar Nova Mulher
1. Crie um arquivo HTML baseado em `katherine-johnson-new.html`
2. Customize as variáveis no `<style>` para as cores
3. Adicione um card em `index-new.html` na seção biografias
4. Ligue com links cruzados

### Modificar Animações
As animações estão em `design/components.css`. Procure por `@keyframes` para editar.

### Desativar JavaScript
Remova `<script type="module" src="animacoes/main-new.js"></script>` do HTML.
